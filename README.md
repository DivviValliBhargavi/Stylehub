# StyleHub — Fashion E-Commerce Website

A modern, responsive fashion e-commerce site built with pure HTML, CSS, and JavaScript.

## 📁 Folder Structure

```
StyleHub/
├── index.html          ← Main HTML (all sections)
├── css/
│   └── style.css       ← All styles, animations, responsive design
├── js/
│   └── main.js         ← Cart logic, product data, scroll effects, form
└── README.md
```

## 🚀 How to Run

Simply open `index.html` in any modern browser — no build step required.

## ✨ Features

- **Navbar** — sticky with scroll-aware shadow, active section highlighting, mobile hamburger
- **Hero** — animated blobs, editorial layout, dual CTAs
- **Marquee** — auto-scrolling announcement strip
- **Women's Fashion** — 8 product cards with SVG illustrations, prices, badges
- **Kids Wear** — 8 product cards with colourful kid-themed illustrations
- **Product Cards** — Add to Bag, wishlist toggle, sale/new badges, hover effects
- **Cart Drawer** — slide-in cart with quantity controls, remove, and total
- **About Section** — stats counter, dual visual cards, reveal animation
- **Contact Form** — validation, success state, toast notifications
- **Footer** — brand, nav columns, social links, legal

## 🎨 Design Tokens

| Token        | Value       | Usage                  |
|-------------|-------------|------------------------|
| `--cream`   | `#faf7f3`   | Page background        |
| `--charcoal`| `#1c1a18`   | Primary text / buttons |
| `--accent`  | `#c8704a`   | CTAs, highlights       |
| `--brown`   | `#8b5e3c`   | Warm accents           |

## 🛠 Tech

- HTML5 semantic markup
- CSS3 — custom properties, grid, flexbox, animations, backdrop-filter
- Vanilla JS — IntersectionObserver, localStorage cart persistence, form validation
- Google Fonts — Cormorant Garamond (display) + DM Sans (body)
